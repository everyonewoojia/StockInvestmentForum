# 软件使用说明书 (User Guide) - 股票基金投资论坛

---

## 1. 产品简介

股票基金投资论坛是一个专为投资者打造的交流社区平台，支持 A 股、港股、美股、基金等投资领域的讨论与分享。

**核心功能**:

- 📝 投资论坛: 发帖、评论、互动、关注
- 🔍 搜索发现: 全局搜索、股票代码自动补全
- 👥 社交系统: 关注、私信、群组
- 🛡 内容审核: 敏感词过滤、管理员审核

## 3. 论坛使用指南

### 3.1 首页浏览

#### 帖子列表 (Feed 视图)
- 默认进入首页后显示所有板块的帖子列表
- **筛选板块**: 点击左侧板块列表可筛选特定板块的帖子
- **帖子卡片**: 显示标题、摘要、作者、时间、互动数据
- **热门排行**: 右侧面板显示最热门的帖子

#### 查看帖子详情
1. 点击感兴趣的帖子卡片
2. 进入帖子详情页，查看完整内容
3. 可看到帖子正文、关联股票、图片附件等
4. 页面下方展示评论列表

---

### 3.2 发帖

1. **前提**: 已登录账号
2. **入口**: 首页Feed视图顶部“发帖”区域
3. **填写内容**:
   - 选择所属板块 (如 A 股市场、基金讨论等)
   - 输入帖子标题 (必填)
   - 输入正文内容 (必填)
   - 可选: 摘要、关联股票代码、上传图片/附件
4. **发布**: 点击“发布”按钮
5. **审核**: 帖子进入待审核状态，管理员审核通过后公开显示

**提示**: 包含敏感词的内容会被系统标记，同样需要管理员审核。

---

### 3.3 评论与互动

#### 发表评论
1. 进入帖子详情页
2. 在底部评论框输入内容
3. 点击“发表评论”
4. 回复他人: 点击评论的“回复”按钮，输入内容

#### 互动操作
- **点赞** : 点击帖子或评论的点赞按钮
- **收藏** : 点击收藏按钮保存帖子
- **分享**: 点击分享按钮
- **取消**: 再次点击可取消互动

---

### 3.4 关注用户

1. 点击用户头像或昵称进入其个人页
2. 点击“关注”按钮
3. 取消关注: 再次点击“已关注”按钮
4. **查看动态**: 切换到“关注”视图，浏览关注用户的发帖

---

### 3.5 搜索

1. 在顶部搜索框输入关键词
2. 按回车或点击搜索按钮
3. **搜索范围**:
   - 帖子: 标题和内容匹配
   - 用户: 昵称匹配
   - 股票: 股票代码和名称匹配
4. **股票建议**: 输入股票代码前缀时自动弹出补全建议

---

### 3.6 群组

#### 浏览群组
- 切换到“群组”视图
- 查看所有公开群组列表

#### 创建群组
1. 点击“创建群组”按钮
2. 填写群组名称、描述
3. 设置可见性和加入策略
4. 创建成功，您自动成为群主

#### 加入群组
1. 在群组列表中选择感兴趣的群组
2. 点击“加入”按钮
3. 公开群组可直接加入

---

### 3.7 私信

1. 切换到“消息”视图
2. 选择联系人或点击“发送消息”
3. 输入接收者用户 ID 和消息内容
4. 点击“发送”

---

### 3.8 个人资料

#### 查看与编辑
1. 切换到“我的”视图
2. 查看个人统计信息 (帖子数、评论数、影响力)
3. 点击“编辑资料”修改昵称、头像、个人简介等

#### 用户认证
1. 在个人资料页点击“提交认证”
2. 选择认证类型:
   - **基础认证**: 自动通过
   - **实名认证**: 填写真实姓名和身份证号
   - **专业认证**: 提供执业证明
3. 认证通过后获得对应徽章

#### 风险测评
1. 在个人资料页点击“风险测评”
2. 完成测评问卷
3. 系统计算您的风险等级 (低/中/高)

---

### 3.9 举报

1. 在帖子详情页或评论旁点击“举报”
2. 选择举报对象类型
3. 填写举报原因和详细说明
4. 提交后管理员会处理

---

## 4. 管理后台 (管理员专用)

**前提**: 使用管理员账号登录 (admin / forum-admin-2026)

### 4.1 进入管理后台
在首页切换到“管理”视图即可访问管理后台。

### 4.2 管理功能概览

#### 仪表盘
- 查看论坛运营数据: 用户数、帖子数、待审核数、待处理举报数
- 热门板块排行和热门帖子排行

#### 内容审核
1. 进入“审核”选项卡
2. 查看待审核的帖子列表
3. 点击帖子查看完整内容
4. 选择“通过”或“驳回” (驳回需填写原因)

#### 举报处理
1. 进入“举报”选项卡
2. 查看待处理的举报
3. 核实后点击“处理”标记解决

#### 用户管理
1. 进入“用户”选项卡
2. 查看用户列表，支持搜索
3. 对违规用户执行: 警告 / 禁言 / 封禁
4. 填写处罚原因和天数

#### 板块管理
1. 进入“板块”选项卡
2. 创建新板块、编辑或禁用已有板块
3. 设置板块名称、别名、分类、市场等

#### 敏感词管理
1. 进入“敏感词”选项卡
2. 查看当前敏感词库
3. 新增或删除敏感词
4. 新增后，帖子/评论包含该词会自动进入审核

## 5. 常见问题 (FAQ)

### Q1: 为什么我发布的帖子看不到？
A: 帖子发布后会进入"待审核"状态，需要管理员审核通过后才会公开显示。请耐心等待管理员审核。

### Q2: 如何修改密码？
A: 目前系统未提供独立的密码修改功能。如有需要请联系管理员。

### Q3: 忘记密码怎么办？
A: 由于目前未实现密码找回功能，请联系管理员重置密码。

### Q4: 搜索不到我想要的内容？
A: 当前搜索基于数据库LIKE匹配，暂未接入Elasticsearch。请尝试使用更精确的关键词。

### Q5: 为什么我收到处罚通知？
A: 您的账号因违反论坛规则被管理员处罚。请联系管理员了解详情。

## 6. 注意事项

1. **内容规范**: 请遵守论坛规则，不要发布违规内容 (如内幕消息、操纵市场、保本收益等)
2. **投资风险**: 论坛内容仅为投资者交流，不构成投资建议，投资有风险，决策需谨慎
3. **数据备份**: 建议定期备份数据库
4. **密码安全**: 请勿使用与其他网站相同的密码，定期更换
5. **隐私保护**: 注意保护个人隐私，身份证号等敏感信息请勿在公开内容中发布
